问题总结
问题一
remote: fatal: early EOF
error: RPC failed; result=55, HTTP code = 200
fatal: The remote end hung up unexpectedly
Writing objects: 100% (114/114), 63.51 MiB | 103.00 KiB/s, done.
Total 114 (delta 12), reused 0 (delta 0)
fatal: The remote end hung up unexpectedl
解决
在修改 家目录下/root/.gitconfig 文件,加入：
[core]
compression = -1

问题二
error: RPC failed; result=55, HTTP code = 200
原因可能是由于网络不好,再git push origin master解决

问题三（服务器仓库存在本地仓库没有的文件）
